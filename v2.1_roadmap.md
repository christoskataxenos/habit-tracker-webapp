# 🚀 PULSE V2.1 Roadmap (Quality of Life Update)
Προγραμματισμένες βελτιώσεις για την επόμενη φάση ανάπτυξης.

---

## 🛠 Φάση 1: System Deep Integration (Windows/Linux)
*Στόχος: Η εφαρμογή να γίνει "ένα" με το λειτουργικό σύστημα.*

- [ ] **Global Hotkeys**
    - [ ] Υλοποίηση `Ctrl + Shift + P` για άμεση εμφάνιση/απόκρυψη του Mini-Focus HUD.
    - [ ] `Ctrl + Shift + S` για Start/Stop χωρίς άνοιγμα του κεντρικού παραθύρου.
- [ ] **System Tray Enhancements**
    - [ ] Προσθήκη Quick Actions στο εικονίδιο της γραμμής εργασιών.
    - [ ] Εμφάνιση εναπομείναντος χρόνου ή κατάστασης (Active/Paused) στο tooltip του tray.

## 📊 Φάση 2: Visual Reporting & Search
*Στόχος: Καλύτερη πρόσβαση και εξαγωγή της πληροφορίας.*

- [ ] **PDF "Impact Report" Engine**
    - [ ] Δημιουργία ενός visual report module.
    - [ ] Εξαγωγή εβδομαδιαίων στατιστικών με γραφήματα σε PDF (χρήση `jspdf` ή παρόμοιο).
- [ ] **Advanced History Search**
    - [ ] Προσθήκη Search Bar στο Timeline Stream.
    - [ ] Φιλτράρισμα βάσει Course, Tag ή ημερομηνιακού εύρους.

## 🎧 Φάση 3: Sensory Experience & UX logic
*Στόχος: Βελτίωση της αίσθησης κατά τη χρήση.*

- [ ] **Focus Audio Cues**
    - [ ] Προσθήκη subtle "Zen" sound effects για Start/Stop/Level-Up.
    - [ ] Επιλογή On/Off στις ρυθμίσεις.
- [ ] **Auto-Pause (Idle Detection)**
    - [ ] Ανίχνευση αδράνειας χρήστη μέσω Electron API.
    - [ ] Prompt για αναδρομική παύση αν ο χρήστης ξεχάσει τον χρονομέτρη ανοιχτό.

## 🧹 Φάση 4: Technical Debt & Cleanup
*Στόχος: Καθαρός κώδικας και βελτιστοποίηση.*

- [ ] **Lint Resolution**
    - [ ] Καθαρισμός όλων των unused imports και variables στο `Dashboard.tsx`.
    - [ ] Διόρθωση των JSX property types στις κλήσεις των components.
- [ ] **Performance Audit**
    - [ ] Έλεγχος memory leaks κατά τη συνεχή χρήση του HUD.
